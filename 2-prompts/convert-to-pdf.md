# HTML PDF Generator Prompt

From the generated result in the markdown output files, create an HTML file with inline CSS (and JavaScript if needed) that displays the content as a PDF-friendly preview with an export button.

Keep it simple but CV-friendly.

## A4 Layout Requirements
- Width: 210mm (A4 width)
- Min-height: 297mm (A4 height)
- `@page` set to A4 size with 15mm margin
- The browser preview and exported PDF should both be A4-friendly
